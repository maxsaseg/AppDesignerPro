Description

Improve your apps with new features:
- Input dialogs and snackbars
- Animation, UI alignment and themes
- Easily create reactive interfaces

System Requirements

MATLAB R2018b or later.

Features

uisnackbar

Fancy snackbars of different types and colors
Documentation:
doc uisnackbar

uiinput

Overlaying input dialog in figure
Documentation:
doc uiinput

uioverlay

Basic overlay panel in uifigure (base for uiinput)
Documentation:
doc uioverlay

uialign

Easily align and distribute UI objects
Documentation:
doc uialign

uitheme

Get colors of predefined themes
Documentation:
doc uitheme

uireactive

Create autoupdating reactive UI strongly binded with data
Documentation:
doc uireactive

uireactivetable

Create autoupdating reactive table strongly binded with data
Documentation:
doc uireactivetable
uiicons
Open Icons App from Icons Toolbox or helps to download toolbox
uiicons

Documentation

To open documentation execute:
uidoc

Examples

To open examples execute:
uiexamples

How to update

To check latest version available execute
UI.ver
To automatically update App Designer Pro to the latest version execute
UI.update
